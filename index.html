Introducing VACCheck

A simple way to check if players in a TF2 match have any VACbans.

Frequently Asked Questions

How to use this site?

  • While in game, open the console and run the status command.
  • Copy the output partially, we only need the user data like so: # 809 "❣г๏zเt๏❣" [U:1:172149372] 29:11 100 0 active. Avoid the headers of the list like # userid name uniqueid connected ping loss.
  • Paste the list into the prompt and click accept.
  • Thats it, the list shows if any player was banned recently.

Why do I need this?

I wanted an easy way to check for hackers, since finding them became something frequent on pubs.
